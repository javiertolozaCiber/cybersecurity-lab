#  Cybersecurity Labs – Portafolio Profesional  
Autor: Javier Toloza  
Carrera: Ingeniería en Ciberseguridad  
Objetivo: Documentar labs reales, análisis de vulnerabilidades y ejercicios prácticos enfocados en seguridad defensiva y ofensiva.

-------------------------------------------

##  Contenido

### 1. Web Security Labs
Evaluaciones a aplicaciones vulnerables:
- DVWA (SQLi, XSS, CSRF)
- OWASP Juice Shop
- WebGoat

Incluye:
- Evidencias
- Controles recomendados
- Matriz de riesgo
- Recomendaciones según OWASP

-------------------------------------------

### 2. Network Security Labs
Ejercicios orientados a:
- Escaneo y enumeración
- Análisis de tráfico
- Firewall rules
- Segmentación de red

-------------------------------------------

### 3. Vulnerability Assessment
Plantillas y reportes profesionales:
- Análisis de vulnerabilidad
- Matriz de riesgo
- Reporte ejecutivo y técnico

-------------------------------------------

### 4. Scripts de Automatización
Scripts de seguridad para tareas comunes:

| Script | Función |
|-------|---------|
| `port_scan.py` | Escaneo rápido de puertos |
| `log_parser.py` | Detección básica en logs |
| `privilege_audit.sh` | Auditoría de usuarios con permisos altos |

-------------------------------------------

##  Objetivo del Repositorio
Demostrar conocimientos prácticos mediante labs reales, aplicando controles, buenas prácticas y documentación profesional utilizada en roles de ciberseguridad.
